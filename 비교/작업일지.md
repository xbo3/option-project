- option-pitch.html, option-full-pitch.html, option-simple.html, option-en.html, option-typing-pitch.html

#### Phase 3: PPT 11슬라이드 (12:30~13:30 KST)
- OPTION-Pitch.pptx (다크 테마, 영어)

#### Phase 4: 모바일 UI 목업 8개 버전 (13:30~15:00 KST)
- premium → light → 2026 → final → accordion → slider → glass-buttons → **complete**
- 글라스모피즘 대칭 버튼, 좌측 카테고리 패널, 10개 정렬탭, 패키지 컬러 체계

#### Phase 5: Railway 배포 (15:00~15:30 KST)
- Express.js + multer 서버, GitHub 연동
- 라이브: https://confident-alignment-production.up.railway.app

#### Phase 6: 샘플 10업체 데이터 + 완성본 (15:30~16:00 KST)
- 겜블시티(gold)/벳콩(blue)/럭키스타/크리스탈벳(purple)/플래티넘/다이아몬드(cyan)/로얄킹/골드러시(orange)/에이스벳/비트윈

#### Phase 7: 로컬 전송 & 라이브 배포 (16:07~17:13 KST)
- dpaste.com → PowerShell 다운로드 → git push → Railway 자동배포

#### Phase 8: 문서화 (17:13~17:40 KST)
- 작업내용.md + 작업일지.md 생성

#### Phase 9: 3D 프리즘 카드 (20:30~20:45 KST)
- public/prism-demo.html — 삼각 프리즘 맞물림 회전
- 왼쪽(시계) ↔ 오른쪽(반시계) 기어 효과
- CSS preserve-3d + rotateY, 4초 자동회전 + 스와이프

#### Phase 10: 프리미엄 쇼케이스 페이지 (21:00~21:30 KST)
- public/showcase.html — 다크 럭셔리 배경 + 골드 파티클 + 폰 목업 2대
- Canvas 80개 입자 상승 애니메이션
- Playfair Display 골드 그라디언트 제목
- 하단 3단계 레이블 + 피쳐 그리드 6개

#### Phase 11: 랭킹 피라미드 캐러셀 (22:00~22:30 KST)
- public/pyramid.html — 1위(중앙대형) → 2-3-4(중단3열) → 5-6-7-8(하단4열)
- 스와이프 시 1위 rotateY(-60deg) 퇴장, 전체 한칸 이동
- GPU transform only, 5초 자동순환

#### Phase 12: 탭-확장 카드 인터랙션 (22:30~23:00 KST)
- public/expand.html — 서브카드 탭 → 화면 중앙 확대 전환
- 폴 축소 + 배너 변형 + 원래 위치 복귀 애니메이션

#### Phase 13: 스플릿 히어로 슬라이드 v1~v3 (23:00~ KST)
- public/slide.html — 1위 카드 = 모바일 프리뷰(좌) + 정보배너(우) 분할
- 모바일 폭 = 서브카드 폭 동일 (33.33%)
- v1: 기본 구현
- v2: DOM 고정 + CSS transform only — 끊김 제거
- v3: **flow 레이아웃** (absolute 제거) — 프레임 미리 잡힘
- 지그재그 동시 이동: ↑←↑← 교대 패턴
- 스와이프/마우스/키보드/5초 자동순환

### 📅 2026년 3월 8일 — 서버 수정 & 전체 배포

#### Phase 14: Express static 경로 수정
- `express.static('public')` → `express.static(path.join(__dirname, 'public'))`
- 상대경로 문제로 Railway에서 index.html 외 static 파일 404 발생
- 수정 후 전체 페이지 200 OK 확인
